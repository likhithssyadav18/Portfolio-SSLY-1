My Projects and Researches

Least Square Support Vector Machine(LSSVM) for Software Defect Forecasting

This project focuses on creating a defect prognosis tool using software metrics. It employs feature selection techniques to optimize metrics for a fault prediction model based on Least Squares Support Vector Machine, validated via a cost-analysis model.

ML Log Analysis and ML Log Privacy Preserving

This project involves analyzing open-source ML pipeline log messages to assess privacy risks using Scala-based ML applications on Apache Spark. By evaluating potential information exposure and offering recommendations, it aims to bolster ML log privacy, contributing to data privacy discussions in the ML community and ensuring user information security in ML systems.

CrystalBall: A Visual Analytic System for Future Event Discovery and Analysis from Social Media Data

This project introduces Crystalball, a visual analysis system pinpointing and ranking future Twitter events using seven parameters from tweets. Five visualizations depict event traits, accompanied by a case study scrutinizing these derived future events from Twitter data.

Fake Review Detection

Using machine learning and deep learning models trained on Yelp's dataset, this project automates fake review detection, addressing imbalanced data, feature engineering, analysis, hyperparameter tuning, and concludes with performance insights and enhancement strategies.

Resting-State Functional MRI(rs-fMRI) analysis using Data Mining techniques

This project employs image processing on rs-fMRI scans, encompassing boundary extraction, supervised classification of IC images into noise or RSN, and unsupervised clustering to detect brain slice clusters.

A Comparitive Study of Performance Metrics of Data Mining Algorithms on Medical Data

Using four data mining algorithms on the ILPD dataset, this research compared six prediction models via 10-fold cross-validation. SVM excelled with 74.82% accuracy, while Naive Bayes lagged behind at 56.55%, with additional metrics detailed in the paper.

Implementing a database application with UI features in MS Access

This project involves creation of database tables for On-site and Off-site users for request and return functionalities along with some features in MS Access.

Automated Warehouse Scenario

This project automates order fulfillment in warehouses using robots, optimizing inventory management and order processing while assigning robots to transport items from storage to designated picking stations for specific orders by using ASP programming.

Enhancing Autonomous Vehicle Safety through Real-time User Health Monitoring and Various factors Integration

This project endeavors to create a safety-driven framework tailored for autonomous vehicles, concentrating on the real-time monitoring of vital passenger health metrics. With a specific focus on intoxication levels, respiratory rate, heart rate, BMI, and sleepiness, the project aims to develop an adaptive advisory control system.

The Comparative Study of LS-SVM and SVM algorithms for Regression and Classification problems

This project conducts a comparative analysis of LS-SVM and SVM algorithms for regression and classification, applying them to benchmark datasets including Breast Cancer Wisconsin and Extreme-Weather Temperature Prediction. Evaluation metrics like accuracy, precision, recall, EV, and RMSE assess their performance.

Data Processing at Scale

The project involves Python functions in a NoSQL setup to find businesses by city and proximity using the Haversine formula. It also executes spatial analysis in Hadoop/Spark on NYC Taxi data, employing Hot Zone and Hot Cell Analysis to enhance location-based services like ride-sharing.
